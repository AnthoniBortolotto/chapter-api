## Description

Repository from the Compass.uol chapter presentation, the repository have examples of the OpenAI API being integrated into a [Nest](https://github.com/nestjs/nest) framework API.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
